# Use Case model

A Use Case Model is an overview of the use cases, like a TODO of sorts.

## Primary Actors

- *User*: k8s api user
- *System*: the kubernetes api
- *End User*: Someone using an application within the k8s cluster

## Use Cases

- U1 - User login
- U2 - User logout
- U3 - User Password Change
- U4 - ...
